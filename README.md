# JavaScript Console Quiz Game

This project is a simple **console-based quiz game** built using **core JavaScript**.  
The quiz runs entirely in the **browser console** and uses `prompt()` for user input and `alert()` for feedback.  
No HTML UI or DOM manipulation is used.
